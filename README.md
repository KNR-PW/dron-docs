# dron-docs

Specjalne repozytorium do dokumentacji wszelkich działań w kole. \
Dokumenty które tutaj umieszczasz będą bezpośrednio odzworowane na stronie: https://knr.edu.pl/dron-docs/

## Jak używać tego repo?

Po pierwsze by móc edytować i tworzyć nowe treści na stronie musisz je pobrać więc jak to zrobić?

### Pobieranie GIT'a

Wejdź na tą [stronę](https://git-scm.com/install/windows) i pobierz najnowszą wersje dla swojego systemu, jeżeli używasz windowsa podczas isntalacji będziesz miał mase opcji do zaznaczenia o których nie musisz się martwić możesz zostawić wszystko na wartości domyślne.

### Pobranie VSCode (opcjonalne)

By łatwiej ci się pracowało z edytowaniem tego projektu zalecane jest posiadanie programu Visual Studio Code. Możesz go pobrać [tutaj](https://code.visualstudio.com/download)

Kiedy masz już podstawowe narzędzia możemy przejść do działania.

### Wersja dla systemu windows
1. Otwórz folder gdzie chcesz pobrać repozytorium
2. Naciśnij prawy przycisk myszy kliknij więcej opcji i wybierz open **git bash here**
3. Pojawi ci się okno z terminalem gdzie musisz wpsiać tą komende:
```bash
git clone https://github.com/KNR-PW/dron-docs.git
```
4. Następnie uruchom program VS Code w tym folderze

Po przejściu tych kroków jesteś gotowy działać z dokumentajcą\
Przejdź teraz na stronę w zakładke **Poradnik** gdzie masz krok po kroku wytłumaczone jak dobrze edytować strone.
