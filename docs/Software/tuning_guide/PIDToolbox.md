# PIDToolbox

**PIDToolbox** to zestaw narzędzi do analizy logów lotu i strojenia regulatorów PID oraz filtracji w dronach. Pozwala wizualizować odpowiedź układu sterowania, analizować szumy żyroskopu, oceniać działanie filtrów (notch, low-pass) i precyzyjnie dobierać nastawy PID.

Twórca narzędzia prowadzi Patreona, na którym za **20 zł/miesiąc** dostarcza ogromną ilość materiałów dotyczących profesjonalnego strojenia PID i filtracji — to zdecydowanie najlepsze źródło wiedzy o profesjonalnym tuningu dronów.
