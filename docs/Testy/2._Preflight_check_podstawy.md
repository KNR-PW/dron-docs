# Podstawowy preflight check list

Poniżej zostały podane kroki które trzeba wykonać przed wykonaniem lotów:

::: danger
Nie wykonanie wszystkich punktów może prowadzić do niepomyślnego i niekontrolowanego lotu UAV
:::

::: tip
Gdy dana jednostka nie przechodzi wszystkich testów lepiej jest przerwać testy by móc naprawić błędy niż reperować całego drona
:::

<label> 1. Sprawdź stan naładowania baterii drona</label>
<input type="checkbox">

<label> 2. Podłącz się z FC za pomocą GCS i sprawdź czy widzi gps (może to potrwać parę minut aż złapie stabilny sygnał)</label>
<input type="checkbox">

<label> 3. Włącz RC, ustaw odpowiedni model na radiu i sprawdź czy FC widzi sygnał z pilota</label>
<input type="checkbox">

<label> 4. Podłącz baterie do drona </label>
<input type="checkbox">

<label> 5. Połącz się z dronem za pomocą bezprzewodowej telemetrii </label>
<input type="checkbox">

<label> 6. Sprawdź czy wszystkie silniki się kręcą, możesz spróbować zarmować drona lub wykonać motor test </label>
<input type="checkbox">

<label> 7. Wykonaj kalibrację kompasu oraz acceli i zrestartuj drona</label>
<input type="checkbox">

<label> 8. Sprawdź czy FC wysyła jakieś błędy do GCS, jeżeli tak spróbuj je rozwiązać</label>
<input type="checkbox">

::: info
Jeżeli udało ci się przejść przez wszystkie punkty gratulacje jesteś gotowy do lotów. Zalecane jest wykonanie równieź listy dotyczącej lotów w trybie GUIDED/OFFBOARD, która znajduje się na następnej stronie.
:::
